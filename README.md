# 3-Axis Magnetic Field Prediction using LSTM

## Overview

This project aims to predict future magnetic field states using a Long Short-Term Memory (LSTM) neural network.

The system consists of:

- ESP32
- Three Hall sensors
- Real-time data acquisition
- Time-series prediction

## Hardware

- ESP32
- Hall Sensors
- Permanent Magnet

## AI Model

- Input: Bx, By, Bz
- Model: LSTM
- Output: Future magnetic field state

## Future Work

- Motion pattern recognition
